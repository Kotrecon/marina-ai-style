# Marina AI Style — TODO

## Бизнес-флоу: Рекомендации с фото

### Текущая проблема

AI получает только текстовое описание вещей. Фото не используются — AI не видит фасон, фактуру, сочетаемость.

### Решение: 3-этапный флоу

#### Этап 1 — Текстовый фильтр (GPT-3.5 / Ollama)

- Вход: погода, сезон, стиль, повод, пол/возраст
- Выход: список ID вещей-кандидатов (10-15 штук)
- Токены: ~200-300

#### Этап 2 — Визуальный анализ (GPT-4o / multimodal Ollama)

- Вход: фото отобранных вещей (10-15 штук, thumbnails 200x200)
- AI анализирует: фасон, фактура, цветовые сочетания, износ
- Выход: финальный подбор 3-5 вещей + совет

#### Этап 3 — Формирование образа

- Вход: финальный список вещей + их фото
- Выход: карточка образа с фото вещей, совет по стилю

### Промпт для Ollama (локальная отладка)

```bash
Система: Ты стилист-стилист. Анализируешь фото одежды и подбираешь образы.
ПРАВИЛА:
- Подбери минимум 2 вещи: верх + низ
- Учитывай цветовые сочетания на фото
- Учитывай фасон и фактуру
- Не рекомендуй вещи с видимыми дефектами
- Формат: [ID] Название — совет

Погода: {city}, {temp}°C, {description}
Сезон: {season}, Стиль: {style}
Повод: {occasion}
Пол: {gender}, Возраст: {age}

Гардероб (фото прикреплены):
[{id}] {name} ({category}, {color})

Верх + низ. Краткий совет по образу.
```

### Техническая реализация

1. Thumbnail-сервис: ресайз фото до 200x200 перед отправкой в AI
2. Кеш thumbnail'ов ( filesystem или Redis )
3. Batch-запрос: все фото в одном message (GPT-4o supports multiple images)
4. Fallback: если multimodal недоступен — текстовый режим (текущий)
